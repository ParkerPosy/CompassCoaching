# SEO Strategy & Implementation

## Overview
Compass Coaching uses TanStack Router's `head()` function on every route to manage SEO metadata. The `__root.tsx` provides fallback meta tags, which child routes override with page-specific content.

> **Canonical domain**: `https://compasscoachingpa.org`
> **SEO constants**: `src/lib/seo.ts`
> **Sitemap generator**: `scripts/generate-sitemap.ts` (runs pre-build)
> **Main SEO doc**: `SEO_BEST_PRACTICES.md` (comprehensive strategy & guidelines)

## Technical SEO Implementation

### Route-Level `head()` Pattern

Every route defines a `head()` function inside `createFileRoute()`. Public pages include full OG, Twitter, and canonical. Private pages use `noindex, nofollow`.

**Public page template**:
```tsx
const HOSTNAME = "https://compasscoachingpa.org";

export const Route = createFileRoute("/example")({
  component: ExamplePage,
  head: () => ({
    meta: [
      { title: "Page Title | Compass Coaching" },
      { name: "description", content: "150-160 char description." },
      { property: "og:type", content: "website" },
      { property: "og:title", content: "Page Title" },
      { property: "og:description", content: "OG description." },
      { property: "og:url", content: `${HOSTNAME}/example` },
      { property: "og:site_name", content: "Compass Coaching" },
      { property: "og:image", content: `${HOSTNAME}/discord-icon.png` },
      { name: "twitter:card", content: "summary" },
      { name: "twitter:title", content: "Page Title" },
      { name: "twitter:description", content: "Twitter description." },
    ],
    links: [{ rel: "canonical", href: `${HOSTNAME}/example` }],
  }),
});
```

**Private page template**:
```tsx
head: () => ({
  meta: [
    { title: "Page Title | Compass Coaching" },
    { name: "description", content: "Description." },
    { name: "robots", content: "noindex, nofollow" },
  ],
}),
```

### Structured Data (Schema.org)

| Schema | Route | Purpose |
|--------|-------|---------|
| WebSite | `__root.tsx` | Site identity, name, URL |
| NonprofitOrganization | `/` (homepage) | Organization details |
| Dataset | `/careers` | PA wage data description |
| Article | `/resources/articles/$slug` | Article metadata, word count, keywords |
| BreadcrumbList | `/resources/articles/$slug` | Navigation breadcrumbs |

### Sitemap

Generated by `scripts/generate-sitemap.ts` as part of the build (`tsx scripts/generate-sitemap.ts && vite build`).

- Static routes with priority/changefreq
- Dynamic article routes auto-collected from `ALL_RESOURCES` (filtered: `active && slug && sections.length > 0`)
- Output: `public/sitemap.xml`

### Title Format
- **Always use `|` as separator** (not `-` or `:`)
- Standard pages: `Page Title | Compass Coaching`
- Extended titles allowed for backlink-opportunity pages (careers, resources)

### Meta Descriptions
- 150-160 characters maximum
- Include primary keyword naturally
- Specific numbers/facts when available
- End with call-to-action
